
# Olist SQL Data Warehouse & Analytics

## Project Overview
Designed and implemented an end-to-end SQL-based data warehouse using the Brazilian Olist e-commerce dataset to support business analytics and executive reporting.

## Key Features
- Built raw, staging, and fact tables using SQL
- Designed star-schema style analytics tables
- Implemented data quality checks (duplicates, NULLs, revenue validation)
- Created analytics-ready views for reporting

## Tech Stack
- SQL (DuckDB)
- Python (Google Colab)
- Pandas

## Key Analytics
- Monthly revenue and order trends
- Top revenue-generating product categories
- Clean fact tables for BI consumption

## Dataset
Public Brazilian E-Commerce Dataset (Olist)
